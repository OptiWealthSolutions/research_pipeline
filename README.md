# Research Pipeline

⸻



## Un pipeline de recherche dédié à l’exploration et la validation d’inefficiences de marché à travers une approche quantitative moderne.
Le projet combine :
	•	Machine Learning appliqué aux séries financières
	•	Alternative data (macro, microstructure, news-based signals…)
	•	Détection et exploitation des inefficiences
	•	Notebooks d’exploration et de prototypage rapide
	•	Modules réutilisables pour backtests, feature engineering et analyse statistique

L’objectif du repository est de fournir un environnement structuré pour :
✅ tester des hypothèses de marché,
✅ construire des signaux prédictifs,
✅ analyser leur robustesse,
✅ et préparer leur implémentation dans un moteur de stratégie.

Un espace de R&D évolutif, centré sur la recherche d’alpha et l’innovation quantitative.
